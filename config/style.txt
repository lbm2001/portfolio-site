PORTFOLIO SITE — VISUAL STYLE REFERENCE
Extracted from app/globals.css. Give this to another agent to reproduce the same look
(fonts, colors, type scale, spacing, component styling) on a different site.

This is a style guide, not a pixel-perfect spec — adapt values to fit the new site's own
layout and content rather than forcing an exact match.

COLORS
  --red      #e12d1a   accent (links-on-hover underline, tags, active/training states, CTA highlights)
  --ink      #111111   primary text / headings
  --body     #4a4a4a   body copy
  --body-2   #5a5a5a   secondary body copy (blurbs, subtitles)
  --muted    #9a9a9a   labels, meta text, placeholders, disabled-ish
  --bg       #ffffff   page background
  --bg-alt   #fcfcfc   alt background
  --border   rgba(0,0,0,0.1)   hairline borders everywhere (1px solid)
  Nav uses a translucent red wash: rgba(225,45,26,0.04) as a sticky-header background with
  backdrop-filter: blur(10px).

FONTS
  --font-sans: "Helvetica Neue", Archivo, Arial, sans-serif   (all UI text, headings, body)
  --font-mono: ui-monospace, Menlo, monospace                 (labels, meta, tags, nav-style numerics)
  Body: antialiased (-webkit-font-smoothing: antialiased; -moz-osx-font-smoothing: grayscale).
  No web fonts loaded — system/Helvetica stack only.

TYPE SCALE (weight / size / line-height, font shorthand style)
  Hero H1:            700 46px/1.02   sans, letter-spacing -0.03em
  Page title:         700 44px/1.04   sans, letter-spacing -0.03em (34px on mobile <720px)
  Section lead (intro para): 400 26px/1.5  sans, letter-spacing -0.015em, max-width 820px
  Work card title:     600 21px/1.22   sans, letter-spacing -0.012em
  Prose h2:            700 22px/1.3    sans, letter-spacing -0.02em
  Prose h3:            700 17px/1.35   sans
  Resume title (name/entry): 600 22px / 16px  sans
  Body copy (prose p/li):    400 16px/1.7    sans
  Blurb / excerpt:     400 14px/1.5–1.55 sans
  Nav links / buttons: 500–600 14px/1    sans
  Small labels (mono, uppercase, tracked out): 600 10–11px  mono, letter-spacing 0.14–0.18em, uppercase
    (used for: .label-mono, hero-tag, resume-section-title, info-cell .k)
  Meta / dates / venues (mono): 500 11–13px mono
  Tags/pills: 500 11px sans

SPACING & LAYOUT
  Page container: max-width 1100px, margin auto, padding 72px 56px 96px (56px 24px on mobile)
  "Narrow" page variant: max-width 760px
  Section block: padding 54px 56px, border-bottom 1px solid var(--border)
  Nav bar: padding 18px 56px, sticky top, z-index above content
  Standard horizontal gutter: 56px desktop -> 24px mobile (breakpoint 720px)
  Grid-of-cards patterns use 0-gap grids with shared 1px hairline borders between cells
  (border-top/border-left on the grid, border-right/border-bottom on each cell) rather than gap+shadow.

BORDERS, RADIUS, EFFECTS
  Hairline borders: 1px solid rgba(0,0,0,0.1) — used constantly instead of shadows/dividers
  Border radius: small and consistent — 3px on buttons/photos-square-crop/tags-are-pill(999px)/inputs
  No box-shadows anywhere in the stylesheet — flat, border-driven design throughout
  Circular avatar: border-radius 50%, 1px border, diagonal-stripe placeholder background
    (repeating-linear-gradient(135deg, #ececec 0 7px, #f6f6f6 7px 14px)) — used before a real photo loads
  Transitions: fast, simple ease (0.2s–0.4s) on opacity/transform/border-color — no bouncy easing

BUTTONS
  Primary: background var(--ink), color #fff, padding ~14px 24px, border-radius 3px, font 600 14px sans
  Outline: transparent bg, color var(--ink), border 1px solid rgba(0,0,0,0.22), same padding/radius/font
  Link-style CTA (detail links, work links): text with a 1.5px solid var(--red) bottom border instead of
  underline — this is the site's signature "link" look, used instead of classic underlines throughout.

TAGS / PILLS
  font 500 11px sans, color #555, border 1px solid rgba(0,0,0,0.12), border-radius 999px, padding 3px 10px

OVERALL AESTHETIC
  Flat, editorial, black/white/red — a single red accent against near-black text and grayscale body copy.
  Monospace is reserved for "meta" information (dates, labels, tags, categories) to visually distinguish
  it from prose, which is always the sans stack. Hairline borders + generous whitespace + no shadows give
  a print/magazine-like structure (grid layouts with shared dividing lines rather than card shadows).
